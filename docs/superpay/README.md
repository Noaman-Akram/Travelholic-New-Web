# SuperPay integration references

This folder keeps the non-secret SuperPay reference files used for the iframe integration:

- `superpay-online-payment-iframe-integration-v1.3.pdf`
- `superpay-card-payment-apis.postman_collection.json`
- `superpay-iframe-url-generator.postman_collection.json`

The merchant portal screenshot is intentionally not committed because it exposes live merchant credentials and the secure hash key. Keep real credentials in `.env.local` locally and in Vercel environment variables for deployed environments.
